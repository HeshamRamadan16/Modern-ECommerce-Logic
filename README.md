# Modern-ECommerce-Logic
# 🛒 متجر تكنو ستور (Modern E-commerce App)

هذا المشروع هو نموذج لمتجر إلكتروني يركز على الوظائف التفاعلية (Functional Logic) وتجربة المستخدم.

## 🌟 ماذا يميز هذا المشروع؟
* **إدارة السلة:** إضافة وحذف المنتجات وحساب الإجمالي لحظياً (Real-time).
* **التنبيهات الذكية:** استخدام مكتبة SweetAlert2 لإظهار رسائل احترافية عند إتمام الطلب.
* **البرمجة الديناميكية:** يتم توليد المنتجات من خلال مصفوفة (Array) في الـ JS لسهولة التعديل.

## 🛠️ لغات البرمجة:
* HTML5 / CSS3 / JavaScript
* Bootstrap 5 (RTL Support)
* SweetAlert2 Library
